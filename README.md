# AnimeKnowledgeGraph

## 开放 API 接口

- Q&A api 接口：`http://api.apassbydreg.work/?question={question_in_chinese}` via `GET / POST`
- 返回值：dict
  - `success`：是否成功
  - `resut`：回答或错误信息（ascii 编码的 utf-8 字符串，可能需要转码）

> 调用样例： http://api.apassbydreg.work/?question=有马公生是哪个番剧里的角色？
